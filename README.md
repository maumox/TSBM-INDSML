# TSBM - Introduction to Data Science and Machine Learning
Welcome to the Introduction to Data Science and Machine Learning course offered to Biomedical Systems and Mechatronics Engineering students at the National Autonomous University of Mexico.

Instructor: MSc. Mauricio Morales

The course emphasizes structured experimentation, feature engineering, and model interpretation. Students work with Python tools including NumPy, pandas, and scikit-learn to explore statistical inference, supervised learning, and unsupervised learning while applying these techniques to real analytical problems.
